## Getting Started: 

- Clone Repo 

```bash
cd /path/to/folder
mkdir calculator
cd calculator
git clone https://github.com/Arvind-4/Calculator-In-PyQt5.git .
```  

- Create a Virtual Environment

```bash
cd calculator
python3.9 -m venv .
source bin/activate
```

**For Windows use:** `.\Scripts\activate`

- Install Dependencies

```bash
pip install -r requirements.txt
```


- Run the App

```bash
python src/main.py
```
